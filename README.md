# Pengen-belajar-2-

A simple news landing page test using the NewsAPI.

## How to Run Locally

1. Open your terminal in this directory.
2. Start a local server (using Python):
   `python -m http.server 8000`
3. **Crucial:** Open your browser and navigate to:
   http://localhost:8000 (Do not use 0.0.0.0)
